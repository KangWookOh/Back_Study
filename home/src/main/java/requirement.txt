출결관리 프로그램

테이블 : Commute(통근), Pause(휴식시간), Member(사원)
Commute column => 사원아이디,출근시간,퇴근시간,휴식시간
Pause column => 사원아이디,정지시간,근무재개시간,휴식사유(흡연,점심,석식,수면실)
Member column => 사원아이디,이름,부서FK(이건 넣고싶으면 하세여 부서 테이블 만들어서)

1. 로그인한 사람의 한 주간의 출결리스트
2. 부서별 하루의 출결 리스트
3. 정상출근 허용시간은 오전 10시, 그 이후에 출근한 사람은 DB에는 지각사유를 적되, 지각 시간은 DB에 들어가지 않아야함.
- 리스트 뿌릴때 LocalDateTime을 활용하여 응답할것.
4. 근무시간은 7시  조기퇴근할 시에 반차,조퇴 사유가 DB에 들어가야함
- 예를들어 출근을 09:30 분에 하였으면 퇴근 가능시간은 +7시간인 17:30분(점심시간 1시간은 근무시간으로 안침) 점심시간도 Pause 타임에 들어가야함
- 한마디로 점심,흡연, 석식 때는 pause라는 버튼을 누르면 사유와 함께 정지시각과 사유가 들어가야하고 근무재개 버튼을 누르면 근무시간으로 허용됨
